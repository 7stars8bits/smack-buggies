Smack Buggies

Warning: This game is filled with bugs!

How to play: When the game begins, use the broom to swat the bugs that appear by clicking on them!

You will have 30 seconds to smack as many bugs as you can. If you aren't fast enough, more bugs will appear!

Each bug you smack is worth one point. How many bugs can you smack before the timer expires?
